Research Experience

LLM Inference Optimization via KV Cache Compression

Nov 2024 - Apr 2025

Final Year Project, The Hong Kong Polytechnic University

Worked on KV cache compression for efficient LLM inference under the supervision of Prof. Zhaorui Zhang and Dr. Xin Yao from Huawei. I evaluated cuSZp compression components, used vLLM to host LLMs, modified FlashAttention interfaces to integrate compression without changing vLLM kernels, and measured perplexity on WikiText-103 under different error bounds.

Outcome: compressed KV cache to 40% of the original size while preserving output quality.

FlexNPU Resource Sharing System

Oct 2025 - Jun 2026

Research Intern, Huawei Hong Kong Research Center, Theory Lab

Worked on a client-server system that intercepts and forwards CANN API calls for elastic NPU resource sharing. I implemented a shared memory transmission model, designed client thread reuse with thread-local slots, and refactored the server-side thread architecture with polling and adaptive worker threads.

Outcome: reduced server CPU load from 300% to 160%, with only 3.87% LLM inference throughput decrease.

Retrieval-Augmented Generation for Embodied Carbon Analysis

May 2024 - Sep 2024

Research Assistant, Faculty of Engineering, The Hong Kong Polytechnic University

Built a domain-specific RAG system for analyzing embodied carbon of computing devices. I processed industry reports with PDF parsing and OCR tools, evaluated embedding models, implemented local vector retrieval, explored distance functions, built hybrid vector-keyword search, and designed a module for carbon value computation.
