About me

I am currently a MSc student in Computer Science at The Chinese University of Hong Kong. I received my BSc (Hons) in Computing with First Class Honors from The Hong Kong Polytechnic University, with a minor in Applied Mathematics.

My research interests are in machine learning systems and computer systems, especially LLM inference optimization, KV cache compression, AI system resource optimization, and retrieval-augmented generation systems.

Research Interests:

LLM Inference, MLSys, and AI Infrastructure:

  • LLM inference optimization: KV cache compression, vLLM-based serving, FlashAttention interface integration, perplexity-quality evaluation, and memory-efficient decoding.
  • AI system resource optimization: NPU resource sharing, CANN API interception, RPC, shared memory transmission, server-side scheduling, and elastic utilization of AI accelerators.
  • RAG and domain-specific QA: document parsing, OCR, embedding models, vector databases, hybrid vector-keyword retrieval, and question answering over industrial reports.

Research Experiences:

LLM Inference Optimization via KV Cache Compression

Final Year Project, The Hong Kong Polytechnic University, Nov. 2024 - Apr. 2025

  • Evaluated cuSZp compression components including quantization, prediction, fixed-length encoding, global synchronization, and block bit-shuffle.
  • Used vLLM to host LLMs and modified FlashAttention interfaces to integrate compression without changing vLLM kernels.
  • Measured WikiText-103 perplexity under different compression ratios and error bounds.
  • Outcome: compressed KV cache to 40% of the original size while preserving output quality.

FlexNPU Resource Sharing System

Research Intern, Huawei Hong Kong Research Center, Theory Lab, Oct. 2025 - Jun. 2026

  • Implemented a shared memory transmission model with global control, slot metadata, and small/large data planes.
  • Designed client thread reuse with thread-local slots and refactored the server-side execution model with polling and adaptive worker threads.
  • Outcome: reduced server CPU load from 300% to 160%, with only 3.87% LLM inference throughput decrease.

Retrieval-Augmented Generation for Embodied Carbon Analysis

Research Assistant, Faculty of Engineering, The Hong Kong Polytechnic University, May 2024 - Sep. 2024

  • Processed industry reports with PDF parsing, OCR, and open-source document extraction tools.
  • Evaluated embedding models, implemented local vector retrieval, explored distance functions, and built hybrid vector-keyword search.
  • Designed a computational module to calculate embodied carbon values for CPU, memory, and storage components.
