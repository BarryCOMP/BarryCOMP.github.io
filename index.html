Selected Work

Research and Engineering Projects

Nov 2024 - Apr 2025

LLM Inference Optimization via KV Cache Compression

Final Year Project · PolyU · vLLM, cuSZp, FlashAttention

Evaluated cuSZp compression components including quantization, prediction, fixed-length encoding, global synchronization, and block bit-shuffle. Modified FlashAttention interfaces to integrate compression without changing vLLM kernels, then evaluated WikiText-103 perplexity under different compression ratios and error bounds.

Outcome: compressed KV cache to 40% of the original size while maintaining output quality.

Oct 2025 - Jun 2026

FlexNPU Resource Sharing System

Research Intern · Huawei Hong Kong Research Center, Theory Lab · NPU, CUDA, RPC, Shared Memory

Implemented a shared memory transmission model with global control, slot metadata, and small/large data planes. Built a thread-local client reuse mechanism and refactored the server execution model with polling and adaptive worker threads.

Outcome: reduced server CPU load from 300% to 160%, with only 3.87% LLM inference throughput decrease.

May 2024 - Sep 2024

Retrieval-Augmented Generation for Embodied Carbon Analysis

Research Assistant · PolyU Faculty of Engineering · OCR, DocX Chain, Embeddings, Vector Databases

Built a domain-specific QA pipeline for embodied carbon analysis of computing devices. Implemented document extraction, embedding evaluation, vector retrieval, keyword search, and a computational module for CPU, memory, and storage carbon values.

Outcome: produced a short publication, DOI 10.1145/3671127.3698699.
